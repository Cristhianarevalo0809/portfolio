Habilidades

Actualmente enfocado en crear sitios web modernos que combinan diseño atractivo y funcionalidad. Utilizo tecnologías como HTML, CSS y JavaScript para desarrollar interfaces responsivas que se adaptan a todo tipo de dispositivos. Trabajo con Git y GitHub para mantener organizados mis proyectos y versionar cada mejora. En este momento estoy profundizando en React y explorando el mundo del diseño web y la experiencia de usuario, con una actitud constante de aprendizaje y superación en cada paso del camino.

HTML

Me defiendo bien con HTML, armo la estructura de las páginas como todo un albañil del código. Desde formularios hasta etiquetas semánticas, lo tengo claro.

CSS3

Con CSS me gusta jugar al diseñador. Me encargo de que las páginas no solo funcionen, ¡sino que se vean bonitas! Manejo Flexbox, Grid y estilos responsive para que todo quede bien en cualquier pantalla.

Javascript

JS es mi compa para darle vida a todo. Hago que los botones respondan, que las tareas se guarden, y hasta le meto algo de lógica para que todo tenga sentido. Aún estoy aprendiendo cosas más complejas, pero me la voy guerreando.

React

Apasionado por construir interfaces dinámicas utilizando React, creando componentes reutilizables que mejoran la experiencia del usuario y optimizan el rendimiento de las aplicaciones. Trabajo con herramientas como React Router, hooks y estados para manejar la lógica de las vistas de manera eficiente. Además, integro proyectos con APIs y gestiono el código con Git y GitHub para mantener un flujo de trabajo ordenado y colaborativo.
